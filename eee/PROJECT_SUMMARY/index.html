    üéØ How to Use

    For Immediate Use:

    1. Extract all files
    2. Install Python and pip
    3. Run: pip install -r requirements.txt
    4. Run: mkdocs serve
    5. Visit: http://127.0.0.1:8000

    For GitHub Pages:

    1. Create GitHub repository
    2. Upload all files
    3. Enable GitHub Pages in settings
    4. Site deploys automatically!

    For Customization:

    • Colors: Edit docs/stylesheets/extra.css
    • Content: Edit .md files in docs/
    • Structure: Modify mkdocs.yml
    • Logo: Add to docs/assets/
